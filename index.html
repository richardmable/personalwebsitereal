Shipping Application

The objective of this project was to create a shipping application that might be used by a shipping company to assign orders, maintain ships, and track routes. The database for this project was the most demanding aspect, taking the better part of two days to accomplish. As one can imagine, there are many, many ways to set up a database for a project like this, depending on how many features one would like to have. Having only 4 days to build the project, we had to learned to compromise, and were very pleased in the result.

BB-8...6 Microblog

A two person group project, this application replicated the general format of Twitter. Challenges for this project included users following each other, limiting the posts to 150 characters, and our first steps towards learning how to validate form information upon submission. It was built with the theme of BB-8, but for bikes, as BB-86 is a bottom bracket type.

Ruby ORM (Recreate Activerecord!)

This project's goal was to recreate some of the widely used features of Activerecord, and to become more familiar with SQL commands. The most challenging part of this project was parsing the strings that were returned by psql. A excellent exercise in logical thinking and how to return data as an object, so that methods may be run on the object.
